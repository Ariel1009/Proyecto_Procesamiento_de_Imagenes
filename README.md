# Proyecto_Procesamiento_de_Imagenes
Proyecto de la materia procesamiento de imagenes
Este proyecto trata de un editor de imagenes como de videos, a estos se les puede aplicar filtros asi como guardarlas con los filtros, ademas del reconocimiento de los dispositivos de video como webcams para 
reconocimiento facial omo el conteo de las personas en pantalla, este proyecto fue creado con C# y .NETCore6
